npmx.dev supports keyboard navigation for faster browsing.

## Global shortcuts

| Key | Action                   |
| --- | ------------------------ |
| `/` | Focus search bar         |
| `?` | Highlight keyboard hints |
| `,` | Open settings            |
| `c` | Open compare             |

::tip
These shortcuts work anywhere on the site. Press `/` from any page to quickly search for a package.
::

## Search results

| Key                       | Action                |
| ------------------------- | --------------------- |
| `Arrow Up` / `Arrow Down` | Move through results  |
| `Enter`                   | Open selected package |

::tip
Type `package@version` (e.g. `vue@3.4.0` or `react@^18`) and press `Enter` to jump directly to that version or filter matching versions.
::

## Package page

| Key | Action           |
| --- | ---------------- |
| `.` | Open code viewer |
| `d` | Open docs        |
| `c` | Compare package  |
